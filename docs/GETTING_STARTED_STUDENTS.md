## Creating Your Account

### Sign Up Options

pi-cast offers two ways to create an account:

#### Option 1: Email Sign Up
1. Click "Sign Up" on the homepage
2. Enter your email address
3. Check your email for a verification code
4. Enter the code to verify your account
5. Complete your profile with your name

#### Option 2: GitHub Sign Up
1. Click "Sign Up with GitHub"
2. Authorize pi-cast to access your GitHub account
3. Complete your profile information

### Setting Up Your Profile

After creating your account:
1. Go to **Settings** → **Profile**
2. Add a profile picture (optional)
3. Write a short bio about yourself
4. Add your location and website (optional)

## Finding Lessons

### Browse by Category
- Navigate to the **Browse** page
- Select a category (Algebra, Geometry, Calculus, etc.)
- Filter by difficulty level (Beginner, Intermediate, Advanced)

### Search for Lessons
- Use the search bar at the top of the page
- Search by lesson title, topic, or teacher name
- Results show matching lessons with previews

### Discover New Content
- **Featured Lessons** - Curated high-quality content
- **Trending** - Popular lessons this week
- **Recommended** - Personalized suggestions based on your activity

## Watching Lessons

### Starting a Lesson

1. Click on a lesson from the browse or search results
2. Read the lesson description and prerequisites
3. Click **Enroll** or **Start Learning**
4. The lesson player will open

### Player Controls

| Control | Function |
|---------|----------|
| ▶️ Play | Start/resume playback |
| ⏸️ Pause | Pause the lesson |
| ⏪ Rewind | Go back 10 seconds |
| ⏩ Forward | Skip ahead 10 seconds |
| 🐌 Speed | Adjust playback speed (0.5x - 2x) |
| 🔊 Volume | Adjust audio volume |

### Using Bookmarks

**Teacher Bookmarks** (shown in blue):
- Pre-marked important moments by the teacher
- Click to jump to key concepts

**Your Bookmarks** (shown in green):
- Create personal bookmarks at any moment
- Add notes to bookmarks for later review
- Access all your bookmarks from the **Bookmarks** tab

### Taking Notes

1. Click the **Notes** button during playback
2. Click **Add Note** at any timestamp
3. Type your note and save
4. Notes are automatically timestamped
5. Click any note to jump to that moment

## Interactive Features

### Manipulating Equations

During lesson playback:
1. Click on any equation block to pause
2. Adjust variable sliders to see changes
3. Watch the graph update in real-time
4. Click **Resume** to continue the lesson

### Variable Sliders

- Slide to adjust variable values
- See immediate visual feedback
- Range: -∞ to +∞ (practical limits apply)
- Type exact values in number inputs

### Graph Interaction

- **Zoom**: Use scroll wheel or zoom buttons
- **Pan**: Click and drag the graph
- **Reset**: Double-click to reset view

## Managing Your Progress

### Dashboard Overview

Access your **Dashboard** to see:
- Enrolled lessons
- Progress on each lesson
- Recently watched content
- Bookmarks and notes

### Tracking Progress

- Progress is tracked automatically as you watch
- Complete bookmarks to increase progress percentage
- View detailed progress in the **Progress** tab

### Resuming Lessons

1. Go to **Dashboard** → **My Lessons**
2. Find the lesson you want to continue
3. Click **Continue Learning**
4. Resume from where you left off

### Watch History

- Access **History** to see all watched lessons
- Organized by: Today, This Week, Older
- Click any lesson to resume
- Remove items from history if needed

## Tips for Success

### 📚 Study Tips

1. **Take Notes Actively**
   - Don't just watch - engage with the content
   - Note down questions for later review
   - Mark confusing parts to revisit

2. **Use Bookmarks Strategically**
   - Bookmark key formulas and concepts
   - Mark sections you need to review
   - Create a personal index of important moments

3. **Experiment with Variables**
   - Pause and adjust equation parameters
   - Observe how graphs change
   - Build intuition through exploration

4. **Manage Your Learning**
   - Set regular study times
   - Complete one lesson before starting another
   - Review bookmarked sections regularly

### 🎯 Best Practices

- **Preview lessons** before deep diving
- **Complete exercises** if available
- **Join study groups** for collaborative learning
- **Ask questions** in lesson discussions (if enabled)

## Need Help?

### Common Issues

| Issue | Solution |
|-------|----------|
| Video won't play | Check your internet connection, refresh the page |
| Audio not working | Check browser permissions, unmute the player |
| Can't create bookmarks | Ensure you're logged in, refresh the page |
| Progress not saving | Wait a moment for sync, check internet connection |
